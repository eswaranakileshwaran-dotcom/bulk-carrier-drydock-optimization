# Bulk Carrier Dry Dock Process Optimization

## Overview
A process improvement study of a Panamax bulk carrier (~75,000 DWT) undergoing
its mandatory 5-year Special Survey dry dock. The project applies Industrial &
Systems Engineering methods to identify bottlenecks, quantify waste, and validate
improvements through simulation.

## Problem
A typical 5-year dry dock runs 25–35 days off-hire. At $15,000–$25,000/day in
lost revenue, poor planning costs $300K–$750K more than an optimized dock.

## Target
Reduce total dock time by 30–40% (from ~30 days to 18–22 days) through process
redesign validated by simulation.

## Methods
- Swim lane process mapping (draw.io)
- Value Stream Mapping — current & future state (Visio)
- Discrete Event Simulation (Arena 16)
- Bottleneck analysis & scenario testing
